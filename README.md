# 🚀 Vishwa Vidhatha | Data Analytics Portfolio

A high-end, recruiter-optimized, and bombastic personal portfolio designed for maximum impact. This project abandons traditional static layouts in favor of a dynamic, app-like landing page built with modern web animation frameworks and interactive 3D elements.

## ✨ Core Features
- **Immersive 3D Engine**: Features a customized [Three.js](https://threejs.org/) interactive particle mesh network background that softly tracks cursor movement.
- **GSAP Scroll Physics**: Integrated [GreenSock (GSAP)](https://gsap.com/) ScrollTriggers so every metric, project card, and timeline block dynamically slides and snaps into view as the user navigates.
- **Dual Display Modes**: Complete support for both "Deep Space Dark Mode" (default) and a clean, premium "Light Mode" via a custom CSS var-toggling engine.
- **Recruiter Optimized Copy**: Employs the CAR (Context, Action, Result) method for all bullet points, strictly bolding high-impact numbers and deployment metrics for instantaneous scanning.
- **Fail-Safe Asset Rendering**: Built-in interactive "Simulation Hubs" ensure that if a live GIF or demo fails to load over a CDN, a sleek, styled console overlay persists to preserve the UI identity.

## 🛠️ Technology Stack
- **Structure**: Semantic HTML5
- **Styling**: Vanilla CSS3 (Custom Properties, Flexbox/Grid Modules, Glassmorphism Variables)
- **Logic & Animation**: Vanilla JavaScript (ES6+), GSAP Core, ScrollTrigger, Three.js

## 💻 Local Testing
1. Clone the repository: `git clone https://github.com/VishwaVidhatha-G/Portfolio.git`
2. Simply open `index.html` in any modern web browser (Chrome, Firefox, Safari).
3. *No local servers or node packages required.*

## 🌐 Live Deployment Guide
To publish this portfolio to the public web for recruiters, you can use any static hosting provider. Here are the two absolute best methods:

### Option A: GitHub Pages (Easiest)
*(This will host your site at `vishwavidhatha-g.github.io`)*
1. In your GitHub account, create a new repository specifically named **`VishwaVidhatha-G.github.io`**.
2. Select "Upload files" and drag-and-drop the 3 files (`index.html`, `style.css`, `script.js`) into the main branch.
3. Commit the changes. 
4. Your website is instantly live! (Sometimes it takes GitHub ~2 minutes to refresh the cache).

### Option B: Vercel (Premium Analytics & Performance)
*(This generates a sleek, fast URL like `vishwa-portfolio.vercel.app`)*
1. Go to [Vercel.com](https://vercel.com/) and create a free account linked to your GitHub.
2. Click **Add New Project**.
3. Select "Import Third-Party Git Repository" and click your Portfolio repository, OR simply drag-and-drop your local `Portfolio` folder directly into the browser dashboard.
4. Vercel will process the HTML instantly and provide you a live, SSL-secured link with an analytics dashboard.
