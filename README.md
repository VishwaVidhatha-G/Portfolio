# Vishwa Vidhatha | Data Analytics Portfolio

A personal data analytics portfolio website created to showcase my projects, experience, and technical skills.

## Built With
- **HTML5 & Vanilla CSS3**
- **JavaScript (ES6+)**
- **GSAP** (for scroll animations)
- **Three.js** (for 3D particle background)

## Local Setup
To run this project locally on your machine:

1. Clone this repository:
   ```bash
   git clone https://github.com/VishwaVidhatha-G/Portfolio.git
   ```
2. Navigate into the directory and open `index.html` in your web browser. No local servers or installations are required.
